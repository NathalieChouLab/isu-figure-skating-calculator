# 🛼 Calculateur de Scores de Patinage Artistique ISU

> Un calculateur de scores entièrement basé sur navigateur pour le patinage artistique compétitif — couvrant le Patinage Simple, les Couples et la Danse sur Glace selon les règles ISU Saison 2025/26.

---

## Ce Que Fait Cet Outil

Cet outil permet aux patineurs, entraîneurs et passionnés de calculer les scores ISU directement dans le navigateur — sans installation, sans serveur, sans compte requis. Il suffit d'ouvrir le fichier HTML et de commencer à calculer.

Il intègre les tableaux officiels ISU de l'Échelle des Valeurs et des Grades d'Exécution tirés des communications :
- **Communication ISU 2707** — Échelle des Valeurs pour le Simple & les Couples
- **Communication ISU 2705** — Échelle des Valeurs pour la Danse sur Glace
- **Communication ISU 2701** — Niveaux & GOE

---

## Fonctionnalités

- **Trois disciplines** — Simple, Couples, Danse sur Glace, chacune avec sa propre liste d'éléments
- **Changement de segment** — Programme Court / Programme Libre (ou Danse Rythmique / Danse Libre)
- **Notation complète des éléments** — Valeur de Base + GOE (−5 à +5) par élément
- **Composantes du Programme (PCS)** — Cinq composantes avec curseurs
- **Déductions** — Chutes, tenue/accessoire, etc.
- **Récapitulatif en temps réel** — TES, PCS, score total mis à jour instantanément
- **Interface en mode sombre** — Design épuré inspiré de la patinoire

---

## Démarrage Rapide

Aucune installation requise.

1. Téléchargez ou clonez ce dépôt
2. Ouvrez `isu_scoring_calculator_fr.html` dans n'importe quel navigateur moderne
3. Sélectionnez votre discipline et votre segment
4. Ajoutez des éléments, réglez le GOE, ajustez les curseurs PCS — votre score se met à jour en temps réel

```bash
git clone https://github.com/NathalieChouLab/isu-figure-skating-calculator.git
cd isu-figure-skating-calculator
open isu_scoring_calculator_fr.html
```
