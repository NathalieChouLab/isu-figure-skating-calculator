# ⛸️ ISU Figure Skating Scoring Calculator

> A fully browser-based scoring calculator for competitive figure skating — covering Singles, Pairs, and Ice Dance under ISU Season 2025/26 rules.

---

## What It Does

This tool lets skaters, coaches, and fans calculate real ISU competition scores directly in the browser — no install, no backend, no account required. Just open the HTML file and start scoring.

It implements the official ISU Scale of Values and Grade of Execution tables from:
- **ISU Communication 2707** — Singles & Pairs Scale of Values
- **ISU Communication 2705** — Ice Dance Scale of Values
- **ISU Communication 2701** — Levels & GOE

---

## Features

- **Three disciplines** — Singles, Pairs, Ice Dance, each with their own element lists
- **Segment switching** — Short Program / Free Skate (or Rhythm Dance / Free Dance)
- **Full element scoring** — Base Value + GOE (−5 to +5) per element
- **Program Component Scores (PCS)** — Five components with sliders
- **Deductions** — Falls, costume/prop, etc.
- **Live score breakdown** — TES, PCS, total score updated instantly
- **Dark-mode UI** — Clean, rink-inspired design

---

## Getting Started

No installation needed.

1. Download or clone this repository
2. Open `isu_scoring_calculator.html` in any modern browser
3. Select your discipline and segment
4. Add elements, set GOE, adjust PCS sliders — your score updates live

```bash
git clone https://github.com/NathalieChouLab/isu-figure-skating-calculator.git
cd isu-figure-skating-calculator
open isu_scoring_calculator.html
```
